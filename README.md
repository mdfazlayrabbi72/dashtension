# Dashtension

Dark new tab dashboard with quick links, dashboard shortcuts, rotating backgrounds, a Pomodoro timer, and notes.

## Setup

1. Open `chrome://extensions` in Chrome.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select the project folder.
4. Open a new tab to view the dashboard.

These steps also work in Chromium-based browsers such as Edge or Brave.

## Permissions

- `storage`: Persist links, dashboard shortcuts, timer settings/state, notes, and background interval.

## Usage

### Links

Add and remove general links from the **Links** panel. These are stored locally and open in new tabs.

### Dashboard

Use the **Dashboard** panel for important service shortcuts (e.g., Cloudflare, Namecheap). Add or remove dashboard links and click the cards to open them quickly.

### Background

Set how often the background changes in minutes. The extension rotates images from curated sources and falls back to an alternate provider if needed.

### Pomodoro

Adjust work/break minutes, then start/pause/reset the timer. Timer settings and current state persist across sessions.

### Notes

Write notes in the **Notes** panel. Content autosaves as you type.
